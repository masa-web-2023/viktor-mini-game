# Anime Guessing Mini-Game

Welcome to the Anime Guessing Mini-Game! In this interactive game, you will be presented with a GIF from an anime, and your challenge is to correctly identify which anime it's from among four provided options.

## Game Details

- This game is powered by the nekos.best API.
- Test your knowledge of anime by visiting the game here: [Anime Guessing Mini-Game](https://masa-web-2023.github.io/viktor-mini-game/)

Can you guess the anime and prove your anime expertise? Enjoy the game and have fun exploring the world of anime!
